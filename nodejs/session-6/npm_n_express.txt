NPM: Node package manager.

How to initialize an app?
npm intit (It will create a package.json file and you can describe the details about your app)

How can I install a package?
npm install <package_name> --save (--save will make an entry into your package.json in dependency list)
--g flag will store the package globally on your system.

How can I install & create app skelton using express?
You need to install "express" & "express-generator"
After that you need to run "express <your_app_name>"
